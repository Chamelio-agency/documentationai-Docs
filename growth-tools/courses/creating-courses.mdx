# Creating and Managing Courses

If you offer coaching programmes, workshops, or educational content, the courses feature lets you create a full learning experience for your clients. They log into their own portal, watch videos, complete lessons, take quizzes, submit assignments, and earn certificates — all under your brand.

## Building your course content

After creating the course, you'll enter the course builder. This is where you add all your content. [Watch this step (3:51)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=231)

### Course structure

Courses are organised in a hierarchy:

- **Course** (the main product, e.g., "8-Week Mindfulness Programme")
  - **Module** / Category (a section, e.g., "Week 1: Foundations")
    - **Lesson** (individual content, e.g., "Introduction to Breath Work")
    - **Quiz** (test their knowledge)
    - **Assignment** (homework they submit to you)

### Adding modules and lessons

Click **Add** to create modules (categories), then add lessons inside them. [Watch this step (5:03)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=303)

<Image
  src="https://loom.com/i/8a9a784706be442a84c8bb78c00dd981?workflows_screenshot=true"
  alt="Course structure showing folders for modules with lessons and quizzes inside"
  width="800"
  height="450"
/>

Each lesson can include:
- **Video** — upload a video walkthrough or embed a Loom/YouTube video
- **Voice notes** — audio content for guided meditations or explanations
- **Written content** — text with formatting, images, and links
- **Downloadable resources** — PDFs, worksheets, checklists, or other files clients can download

### Adding quizzes

Quizzes inside courses work differently from the standalone quizzes. These are knowledge checks — each question has a correct answer. [Watch this step (9:04)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=544)

<Image
  src="https://loom.com/i/4a0f3d491da3430aa10ab135bcc774de?workflows_screenshot=true"
  alt="Course quiz builder showing a question with answer options and correct answer selection"
  width="800"
  height="450"
/>

You can:
- Create multiple-choice questions with a correct answer
- Set a **passing score** (e.g., 80% correct to pass)
- Add explanations for each answer so clients learn from their mistakes
- Require passing the quiz before unlocking the next module

### Adding assignments

Assignments let clients submit their own work for you to review — homework, reflections, or projects. [Watch this step (9:04)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=544)

You provide instructions and optional reference material, and clients upload their work directly in the portal. You review it before they can move to the next lesson.

## Controlling the learning path

You have powerful options for how clients move through your course:

<ExpandableGroup>
  <Expandable title="Locking content" default-open="false">
    You can lock modules or lessons so they only become available after the previous one is completed. This ensures clients follow the course in order instead of skipping ahead. For example, "Module 3 unlocks only after Module 2 is completed."
  </Expandable>
  <Expandable title="Drip content" default-open="false">
    Instead of making everything available at once, you can "drip" content over time. For example, "Module 2 unlocks 14 days after the client enrols." This is perfect for programmes where you want clients to pace themselves — like releasing one week of content at a time.
  </Expandable>
  <Expandable title="Quiz gates" default-open="false">
    Require clients to pass a quiz before accessing the next module. Set a minimum passing percentage (e.g., 80%) to ensure they've understood the material.
  </Expandable>
  <Expandable title="Assignment review" default-open="false">
    Require your review and approval of a submitted assignment before the client can proceed. This gives you direct oversight of their progress.
  </Expandable>
</ExpandableGroup>

<Callout kind="tip">
  For most wellness programmes, a combination of drip content and locking works well. Release one module per week so clients don't get overwhelmed, and lock each module until the previous one is completed so they stay on track.
</Callout>

If you create a design you love, save it as a template so you can reuse it for future courses. [Watch this step (14:00)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=840)

## Course details

The **Details** tab is where you edit the course's metadata: thumbnail, instructor name, description, and advanced settings like tracking codes. [Watch this step (10:54)](https://www.loom.com/share/03623edebec946ff8dc252a8eb18d6be?t=654)

You can also add instructor information — your name, photo, and bio — so clients know who's teaching them.

## Course ideas for wellness businesses

| Course type | Structure | Pricing model |
| --- | --- | --- |
| 8-Week Mindfulness Programme | 8 modules (1 per week), video lessons + guided meditations | One-time or recurring |
| Yoga Teacher Training Prep | 6 modules with quizzes and assignments | One-time |
| Retreat Pre-Arrival Guide | 3 modules: what to pack, what to expect, preparation exercises | Free (included with retreat) |
| Monthly Wellness Membership | New content dripped monthly, community access | Recurring (monthly) |
| Self-Paced Coaching Programme | 12 lessons, unlock at own pace | One-time |
| Free Taster / Lead Magnet | 3 short lessons showing your teaching style | Free |

## Quick reference

| What you want to do | Where to find it |
| --- | --- |
| Create a new course | **Memberships → Courses → Products → Create** |
| Add a module/category | Inside the course → **Add Module** |
| Add a lesson | Inside a module → **Add Lesson** |
| Add a quiz or assignment | Inside a module → **Add Quiz** or **Add Assignment** |
| Lock content in order | Lesson settings → **Lock** → select prerequisite |
| Drip content over time | Module settings → **Drip** → set number of days |
| Set up a certificate | Course → **Credentials** tab |
| Customise appearance | Course → **Customization** tab |
| Save design as template | Customization → **Save as Template** |
| Configure portal settings | **Client Portal → Settings** |
| Preview the course | **Preview** button (top right of course builder) |
