# Creating and Managing Forms

Forms are how people give you their information — whether it's a contact form on your website, an intake questionnaire before a session, a retreat sign-up form, or a payment form. Every submission automatically creates a contact in your system, so nothing falls through the cracks.

### Adding fields

Use the **Quick Add** feature to add fields to your form. Just drag a field and drop it where you want it. [Watch this step (3:05)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=185)

<Image
  src="https://loom.com/i/6f60100d420e4bf5b8d1df5944454abe?workflows_screenshot=true"

Common fields for wellness businesses:

| Field type | Use it for |
| --- | --- |
| Full Name | Every form needs this |
| Email | Primary way to reach the contact |
| Phone Number | For SMS reminders or follow-up calls |
| Text Area | Open-ended questions ("What brings you to this retreat?") |
| Dropdown | Multiple choice ("How did you hear about us?") |
| Date Picker | "What's your preferred start date?" |
| File Upload | Documents, health forms, or photos |
| Checkbox | Consent checkboxes, terms & conditions |
| Address | Shipping or location-based services |

<Callout kind="tip">
  Keep forms short. Every extra field reduces the chance someone completes it. For a contact form, name + email + one question is usually enough. For intake questionnaires, you can be more detailed since the person is already committed.
</Callout>

## Smart form features

### Conditional logic — show and hide fields

This is one of the most powerful features. You can show or hide fields based on what someone answers. [Watch this step (1:59)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=119)

<Image
  src="https://loom.com/i/4c1836787407436ab5844ba3af10fe54?workflows_screenshot=true"
  alt="Show/hide conditional logic setup — if answer is yes, show the next field"
  width="800"
  height="450"
/>

**Example:** You ask "Do you have a website?" If they click **Yes**, a new field appears asking for the URL. If they click **No**, that field stays hidden. The form stays clean and only shows what's relevant.

### Conditional logic — redirect and custom messages

Beyond show/hide, you can set up more advanced rules: [Watch this step (1:01)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=61)

<Image
  src="https://loom.com/i/4fddad2cc0184f7290177c3668b2903c?workflows_screenshot=true"
  alt="Conditional logic options showing redirect, custom message, show/hide, and disqualify"
  width="800"
  height="450"
/>

- **Redirect** — send someone to a different page based on their answer (e.g., different retreat options)
- **Custom message** — show a specific message based on what they selected
- **Disqualify** — if someone doesn't meet your criteria, show them a polite message instead of continuing the form

### Disqualifying leads

If you only work with a specific type of client, you can use the disqualify feature to filter people out. [Watch this step (1:35)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=95)

<Image
  src="https://loom.com/i/74ba545097d9494fa07f3200b1a32cc9?workflows_screenshot=true"
  alt="Lead disqualification setup with custom message for people who don't meet criteria"
  width="800"
  height="450"
/>

For example, if your retreat is only for people over 18, you can show a friendly message to younger visitors instead of letting them complete the booking. This saves everyone's time.

## Notifications

Set up who gets notified when someone fills out the form. [Watch this step (2:23)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=143)

<Image
  src="https://loom.com/i/20a591c8b82a4b01812236352e14ae6a?workflows_screenshot=true"
  alt="Notification settings showing email alerts for form submissions"
  width="800"
  height="450"
/>

You can set up two types of notifications:

- **Notification to you** — get an email every time someone submits the form, so you never miss a lead
- **Autoresponder to the person** — automatically send them a confirmation email after they submit ("Thanks for your inquiry, we'll be in touch within 24 hours")

<Callout kind="tip">
  Always set up a notification for yourself. Even if you have automations handling follow-ups, it's good to know when new leads come in so you can personally reach out to the most promising ones.
</Callout>

## Form settings

The settings tab controls what happens after someone submits the form. [Watch this step (4:56)](https://www.loom.com/share/f083617fc76d44f7a9da928d6b0553cb?t=296)

<Image
  src="https://loom.com/i/cbb0614c5f3f47a28fcc1ca178d7b0a6?workflows_screenshot=true"
  alt="Form settings showing submission actions like redirect, thank you message, and order confirmation"
  width="800"
  height="450"
/>

Options include:

- **Show a thank-you message** — display a custom confirmation message after submission
- **Redirect to a URL** — send them to a specific page (like a booking confirmation or a "next steps" page)
- **Order confirmation** — if this is a payment form, show the order details
- **Facebook Pixel tracking** — track form submissions for your ad campaigns
- **Trigger an automation** — automatically start a follow-up sequence when the form is submitted. [Learn about automations](/growth-tools/automations/understanding-workflows)

## Form ideas for wellness businesses

| Form type | What it collects | When to use it |
| --- | --- | --- |
| Contact form | Name, email, message | Website "get in touch" page |
| Retreat application | Name, email, experience level, dietary needs, goals | Retreat sign-up page |
| Intake questionnaire | Health history, goals, preferences, emergency contact | Before a first session |
| Feedback form | Rating, what they enjoyed, suggestions | After a retreat or coaching programme |
| Newsletter sign-up | Name, email | Website footer, blog posts, social media |
| Discovery call booking | Name, email, "What do you need help with?" | Booking page |

## Quick reference

| What you want to do | Where to find it |
| --- | --- |
| Create a new form | **Sites → Forms → Create New Form** |
| Add fields | **Quick Add** panel (left side of builder) |
| Set up conditional logic | **Conditional Logic** (left panel) |
| Style the form | **Styles & Options** (right panel) |
| Set submission actions | **Settings** (top of builder) |
| View responses | **Submissions** tab (top of builder) |
| Embed on website | **Integrate → Embed Code** |
| Share as a link | **Integrate → Share** |
| Check mobile view | **Mobile mode** toggle (left panel) |
